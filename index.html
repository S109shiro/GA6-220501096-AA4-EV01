Tutorial: Cómo instalar Python en Windows

  1. Descargar el instalador
  2. Instalar Python
  3. Comprobar la instalación

1. Descargar el instalador

Primero debemos entrar a la página oficial de Python y luego hacer clic en la opción “Downloads” para descargar la versión más reciente de Python. En este caso, la versión 3.14.0.

2. Instalación del archivo .exe

Ahora abrimos el instalador que acabamos de descargar. Nos mostrará la siguiente pantalla.

Lo dejamos como se ve en la imagen y hacemos clic en “Install Now”. Comenzará la instalación de todos los componentes necesarios para que Python funcione correctamente en nuestro PC.

Una vez finalizada la instalación, se mostrará esta pantalla indicando que todo fue un éxito. Con esto ya podemos cerrar el instalador.

3. Comprobar que Python esté instalado

Para verificar que Python se haya instalado correctamente, abrimos la consola. Usamos la combinación de teclas Windows + R y escribimos cmd.

Esto abrirá la consola de comandos. En ella escribimos python --version para comprobar si Python está instalado y qué versión tenemos.

Si todo salió bien, ya podemos comenzar a programar desde la consola usando el comando python seguido de las instrucciones que deseemos ejecutar.
